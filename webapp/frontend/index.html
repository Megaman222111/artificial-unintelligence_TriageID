NFC User ID

Scan a card to look up user info, or add a user and write their ID to a card (Arduino 115200 baud)

Scan card

Place a card on the reader. User info is loaded from the server.

Enter user details and choose a User ID (max 15 chars). Then write that ID to an NFC card.

Serial (debug)

Uses Web Serial API. Works in Chrome and Edge. Connect with USB and grant port access when prompted.
